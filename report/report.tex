\begin{abstract}
We scale KV Self-Compaction from a 50M-parameter GPT model (Phase 1) to Qwen3-0.6B-Base
with LoRA finetuning on the Dolci SFT mixture. The method processes sequences in
512-token blocks, appending 64 learned compaction tokens per block whose KV states carry
compressed history to the next block. A learnable per-head attention bias controls
attention to compressed state. Trained with BPTT (K=8), the compaction model achieves
val\_ppl=3.652, within \textbf{12\% of the full-context upper bound} (3.260), using only
64 compressed KV pairs per block instead of the full 4096-token context. An ablation
with K=1 (no temporal gradient) shows val\_ppl=4.139, nearly identical to the isolation
baseline (4.158), demonstrating that BPTT is the critical ingredient---without it, the
model cannot learn to write useful cross-block state.
\end{abstract}

\section{Method}

\subsection{Architecture \& Training}
\begin{itemize}
    \item \textbf{Base model}: Qwen3-0.6B-Base (28 layers, 16Q/8KV heads, head\_dim=128)
    \item \textbf{Adaptation}: LoRA rank=32, $\alpha$=64 on all 7 linear layers (3.28\% trainable)
    \item \textbf{Compaction}: W=512, P=64, K=8 (BPTT depth)
    \item \textbf{New parameters}: compaction\_embeddings (64$\times$1024) + compact\_attn\_bias (16 scalars)
    \item \textbf{Data}: Dolci Think-SFT + Instruct-SFT (50K each, $\sim$95K after filtering)
    \item \textbf{Training}: 600 steps, lr=$10^{-4}$, WSD schedule, total batch$\approx$528
    \item \textbf{Compute}: 3 nodes $\times$ 4 A100-80GB, manual gradient all-reduce DDP
\end{itemize}

\section{Analysis}

\subsection{BPTT is the Critical Ingredient}

The most striking finding is that K=1 $\approx$ E:
\begin{itemize}
    \item K=1 val\_ppl = 4.139, E val\_ppl = 4.158 (0.5\% difference)
    \item K=1 passes compact\_kv between blocks but E does not
    \item Yet both perform nearly identically
\end{itemize}

This means that without temporal gradient (BPTT), the compact\_kv state carries
essentially no useful information. The model can \emph{read} from compact\_kv (the bias
drifts to $-1.24$, showing it attends to compact\_kv positions) but it cannot learn to
\emph{write} useful information into compact\_kv without the training signal from future
blocks.

With K=8 (full BPTT across the 8-block sequence), the model receives gradient from
block $n+1$'s loss flowing through compact\_kv back to block $n$'s compaction embeddings.
This temporal signal teaches the model: ``the compact\_kv you produced in block $n$ was
good/bad for predicting block $n+1$.'' Without this signal (K=1), the compaction
embeddings learn only from within-block loss, which doesn't incentivize useful
cross-block compression.

\subsection{Gap to Full Context}

B's val\_ppl (3.652) is 12\% worse than A's full\_ctx\_ppl (3.260). This gap represents
the information lost by compressing each 512-token block into 64 KV pairs. The gap
is consistent across training (not closing further after step 400), suggesting a
fundamental compression limit at this P/W ratio (64/512 = 1:8 compression).

\subsection{Cross-Block PPL}

The cross\_block\_ppl metric (PPL on the first 32 tokens of blocks 1+) shows even
larger differences:
\begin{itemize}
    \item B: 4.59 --- compact\_kv provides strong cross-block context
    \item K=1: 9.70 --- compact\_kv present but uninformative (2.1$\times$ worse)
    \item E: 12.96 --- no cross-block context at all (2.8$\times$ worse)
\end{itemize}

K=1's cross\_block\_ppl (9.70) is notably better than E's (12.96), suggesting that
even without BPTT, the shared compaction embeddings learn \emph{some} useful compression
through within-block gradients. But this indirect learning is far less effective than
explicit temporal gradient.

\section{Conclusion}

KV Self-Compaction scales to 0.6B-parameter models, achieving within 12\% of
full-context performance using only 64 compressed KV pairs per 512-token block.
The critical finding is that BPTT (K=8) is essential---without temporal gradient,
compact\_kv is nearly useless despite the model learning to attend to it. This
suggests that the compaction tokens must be explicitly trained to carry cross-block
information through a temporal training signal, rather than learning it as a
side-effect of within-block optimization.
